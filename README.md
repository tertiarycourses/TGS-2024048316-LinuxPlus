These are the official hands-on lab exercises for the WSQ CompTIA Certified Linux+ Training course delivered by [**Tertiary Infotech Academy Pte Ltd**](https://www.tertiarycourses.com.sg/).

A complete set of **30 step-by-step labs** aligned to the CompTIA Linux+ XK0-006 (V8) exam objectives. Every lab runs on the free **Killercoda Ubuntu Playground** (https://killercoda.com/playgrounds/scenario/ubuntu) — no local install, no virtual machine, no credit card required.

Alongside the labs, this repo generates the full WSQ **courseware set** — slide deck, Lesson Plan, Learner Guide and a Markdown mirror — all from a single source so they stay 100% aligned to the labs and exam domains.

### How the courseware is built (single source of truth)

The courseware is generated by the WSQ build skills under [.claude/skills/](.claude/skills/), driven by one content module so every artifact stays in sync:

- **`wsq-learner-guide/course_content.py`** — the single source: course metadata, the five exam domains (with weightings and sub-objectives), and the 30 labs (loaded from `courseware/data/*.json`, extracted from [labs/](labs/)).
- **`wsq-slides/build_slides.py`** → the `.pptx` deck.
- **`wsq-lesson-plan/build_lesson_plan.py`** → the Lesson Plan `.docx`.
- **`wsq-learner-guide/build_learner_guide.py`** → the Learner Guide `.docx` **and** `LEARNER-GUIDE.md`.

Regenerate everything after editing the content module:

```bash
python3 .claude/skills/wsq-learner-guide/build_learner_guide.py
python3 .claude/skills/wsq-lesson-plan/build_lesson_plan.py
python3 .claude/skills/wsq-slides/build_slides.py
```

## How to use

1. Open the Killercoda playground in your browser: https://killercoda.com/playgrounds/scenario/ubuntu
2. Pick a lab from the list below and follow the steps in order.
3. Reset the playground between labs that change kernel, firewall, or systemd state.
4. See [labs/tools.md](labs/tools.md) for every free tool used (with install commands and download links).
